## Requirements
- [Nodejs](https://nodejs.org/en/download/).

## Run

> using Git

```sh

# Clone this repository
git clone <https://github.com/karenyov/my-finances.git>

# cmd in project file
# install all dependencies
npm install

# run aplication
npm run dev
```

URL API: http://localhost:3000
